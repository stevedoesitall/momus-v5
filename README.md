# Momus.io v5

## Fullstack web app for Momus.io. Written in TypeScript.

### Frontend:

- Svelte.js
- Bulma.io CSS framework

### Backend:

- Node.js
- Express.js

### Database:

- PostgreSQL
- Prisma.io ORM

### Tooling:

- ESLint for linting
- Prettier for formatting
- Mocha for testing
- ESBuild for bundling
- PurgeCSS for CSS minification
